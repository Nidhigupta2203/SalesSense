# SalesSense  
### Intelligent Sales Forecasting using Time Series & Machine Learning

SalesSense is an end-to-end **sales forecasting system** that analyzes historical sales data and predicts future demand using **time series feature engineering**, **machine learning models**, and an **interactive Streamlit dashboard**.
